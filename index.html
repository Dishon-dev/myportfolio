Expert Finder

Expert Finder is a web based site that securely connects clients with highly competent & qualified experts in nearly all fields based on their location. It uses PHP for server-side scripting and a MySQL database to store information about the experts and their services. It incoporates filtering capabilities based on location enabling clients to search for experts in their geographical location.

Oppi Hospital Management System

This is a hospital management system that manages hospital operations. It allows monitoring of doctors attendance by keeping track of check-in and check-out time. The system allows for patient to book check-up appointments with doctors and keeps track of patient records. The administartor can add and remove doctors and can also assign duties to doctors. The system can generate reports.

Lawyer Finder

Lawyer Finder is an android based application that bridges the gap between lawyers and clients. It computerizes lawyers search with filtering capabilities based on the location and specialization and also providing lawyer real-time location through Google map API integration. The application also allow clients to book appointments with lawyers. It is built using Java and Firebase for backend.

Empower Studio

Empower Studio is a web based service that allows artists/musicians to book for studio sessions. The site is built using HTML, CSS & JavaScript for frontend, PHP for the backend and MySQL for Database. The studio owner through the administartor dashboard can manage the appointments and schedule sessions by contacting the clients.
